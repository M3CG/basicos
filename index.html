Cñorita Nahi, tu tarea

Escribir un programa que:

  1. guarde la cadena "Hola Elnestito!" en una variable y luego muestre el contenido de esa variable por pantalla.

  2. tenga 3 variables numericas y muestre en pantalla la suma de ellas.

  3. De 2 numeros, saber cual es el mayor.

  4. al cargar el precio de un producto, devuelva el precio final del mismo (producto + iva).

  5. De 3 numeros, saber cual es el mayor.

  6. Se cargué el nombre de un gusto de helado en una variable y muestre por pantalla un mensaje que aclare lo rico que es el helado de ese gusto.

  7. Diga si 2 numeros cargados en variables son iguales o diferentes.

  8. Muestre el promedio de 3 numeros.

  9. Diga si un numero guardado en una variable es par o impar.

  10. Tenga en una variable una temperatura en escala Fahrenheit (debe permitir decimales) y le muestre el equivalente en grados Celsius. La fórmula de conversión que se usa para este cálculo es: celsius = (5/9) * (fahrenheit-32)
